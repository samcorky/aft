Keyboard Shortcuts

Use these keyboard shortcuts for faster card creation:

  • N - Add a new card at the top of a column
  • M - Add a new card at the bottom of a column

To use these shortcuts:

  • Hover your mouse over the column where you want to add the card, then press the shortcut key
  • After adding a card, press the shortcut again to quickly add another card to the same column
  • Shortcuts won't work while typing in text fields or when a modal is open

Working Style

AFT supports different working styles to accommodate various team preferences and workflows. You can configure your preferred working style in the General Settings.

Available Working Styles

There are two main working styles available:

  • Kanban - Traditional column-based workflow where cards move through stages (To Do, In Progress, Done, etc.)
  • Board as Task Category - Boards represent work categories, and columns represent task statuses or subcategories within those boards

Done/Not Done Functionality

When using Board as task Category working style, cards display a Done button (checkbox) that allows you to mark individual cards as complete without moving them. This provides quick task completion tracking:

The Done button provides:

  • Quick visual indicator of task completion status
  • Ability to mark tasks complete without moving cards between columns
  • Dynamic column count updates showing total cards and completed cards

Done View

A dedicated "Done View" is available that displays only the cards you've marked as done across all columns. This view helps you:

  • Quickly review completed work
  • Track progress and accomplishments
  • Filter out incomplete tasks to focus on what's finished
  • Access the view from the views dropdown in the header next to the board view options

Column Count Updates

When using the Done button functionality, column headers display updated counts:

  • Total Count - Total number of cards in the column (all cards regardless of done status)
  • Done Count - Number of cards marked as done in the column (shown in parentheses)
  • For example: "To Do (2/5)" means 5 total cards with 2 marked as done
